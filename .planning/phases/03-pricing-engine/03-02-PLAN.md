<feature>
  <name>Pricing Calculation Engine (7 ACMI Components)</name>
  <files>fastapi-project/app/pricing/service.py, fastapi-project/tests/test_pricing_service.py, fastapi-project/tests/test_pricing_fixtures.py</files>
  <behavior>
    ## Component A (Aircraft)
    - aircraft_cost = lease_rent_usd + six_year_check_usd + twelve_year_check_usd + ldg_usd + epr_rate(cycle_ratio, environment) + llp1_rate + llp2_rate + apu_rate
    - EPR rate looked up from matrix by cycle_ratio and environment (benign/hot) with linear interpolation
    - All USD rates converted to EUR using exchange_rate
    - Result = total_aircraft_eur / MGH (per block hour)
    - Note: EPR matrix stores per-engine rates; multiply by 2 (engine count) per Phase 2 decision

    ## Component C (Crew)
    - Crew composition lookup by (aircraft_type, lease_type): pilots, senior attendants, regular attendants
    - A320 wet: 2 pilots, 1 senior, 3 regular. A321 wet: 2 pilots, 1 senior, 4 regular.
    - Damp: pilots only (2,0,0). Moist: pilots + 1 senior (2,1,0).
    - Fixed = (pilot_salary * pilots + senior_salary * senior + regular_salary * regular) * crew_sets + training_budget/fleet + uniform_budget/fleet
    - Variable = per_diem_rate * total_crew_count * crew_sets + accommodation_budget/fleet
    - Result = (Fixed + Variable) / MGH

    ## Component M (Maintenance)
    - Fixed = line_maintenance + base_maintenance + personnel_salary + c_check + maintenance_training (all monthly rates)
    - Variable = spare_parts_rate * MGH + maintenance_per_diem
    - Result = (Fixed + Variable) / MGH

    ## Component I (Insurance)
    - insurance_eur = insurance_usd * exchange_rate
    - Result = insurance_eur / MGH

    ## Components DOC, Other COGS, Overhead
    - DOC = doc_total_budget / average_active_fleet / MGH
    - Other COGS = other_cogs_monthly / MGH
    - Overhead = overhead_total_budget / average_active_fleet / MGH

    ## Margin and Final Rate
    - total_cost_per_bh = sum of all 7 components
    - final_rate = total_cost_per_bh / (1 - margin_percent) when margin > 0
    - revenue_per_bh = final_rate (the billing rate)
    - monthly_revenue = final_rate * MGH
    - monthly_cost = total_cost_per_bh * MGH
    - monthly_pnl = monthly_revenue - monthly_cost

    ## Decimal Precision
    - Every intermediate and final value must be Decimal, never float
    - isinstance(result, Decimal) must be True for all returned values
    - assertEqual (exact), not assertAlmostEqual

    ## EPR Interpolation Edge Cases
    - Exact match in matrix: return that row's rate
    - Between two rows: linear interpolation
    - Below minimum: clamp to first row's rate
    - Above maximum: clamp to last row's rate
    - Empty matrix: raise ValueError

    ## Multi-MSN Aggregation
    - calculate_project_pnl takes list of MSN results and aggregates
    - Total = sum of individual monthly costs and revenues across MSNs
    - Total per-BH rates = weighted average by MGH
  </behavior>

Create fastapi-project/app/pricing/service.py as a pure-function module:

Use dataclasses (not Pydantic) for internal calculation types:
- AircraftCosts: all rate fields from aircraft_rates table
- PricingConfig: all fields from pricing_config table
- CrewConfig: all fields from crew_config table

Implement these functions:
1. interpolate_epr(matrix: list[tuple[Decimal, Decimal]], target_cr: Decimal) -> Decimal
   - Uses bisect.bisect_left for O(log n) lookup
   - Clamps to boundaries, interpolates between rows
2. _calc_aircraft(inputs, aircraft_costs, exchange_rate) -> Decimal
   - Sums all aircraft cost components, converts USD to EUR, divides by MGH
3. _calc_crew(aircraft_type, lease_type, crew_sets, crew_config, mgh) -> Decimal
   - Uses CREW_COMPOSITION dict for (type, lease_type) -> {pilots, senior, regular}
4. _calc_maintenance(mgh, config) -> Decimal
   - Fixed + Variable, divided by MGH
5. _calc_insurance(config, exchange_rate, mgh) -> Decimal
6. _calc_doc(config, mgh) -> Decimal
7. _calc_other_cogs(config, mgh) -> Decimal
8. _calc_overhead(config, mgh) -> Decimal
9. calculate_pricing(inputs, aircraft_costs, pricing_config, crew_config) -> ComponentBreakdown
   - Orchestrates all 7 components + margin calculation
10. calculate_project_pnl(msn_results: list) -> dict
    - Aggregates multiple MSN results into project totals

All arithmetic with Decimal. All internal functions prefixed with _ (private). Module-level CREW_COMPOSITION dict.

Document Excel cell references in code comments for each formula (e.g., "# Source: Cost Forecast sheet, row 15").
