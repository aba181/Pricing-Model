Endpoints:

1. **POST /pricing/calculate** (Depends(get_current_user)):
   - Accepts CalculateRequest body
   - For each msn_input: fetch aircraft by MSN (AircraftRepository), fetch EPR matrix, fetch current pricing_config, fetch crew_config for aircraft_type
   - Build dataclass inputs, call calculate_pricing() from service.py
   - Return CalculateResponse with per-MSN results and project total

2. **GET /pricing/config** (Depends(get_current_user)):
   - Returns current pricing config (PricingConfigResponse)

3. **GET /pricing/config/{version_id}** (Depends(get_current_user)):
   - Returns specific config version

4. **PUT /pricing/config** (Depends(require_admin)):
   - Accepts UpdatePricingConfigRequest
   - Calls repo.create_version() (append-only -- CONF-02, CONF-03)
   - Returns new version

5. **GET /pricing/crew-config** (Depends(get_current_user)):
   - Returns list of current crew configs (repo.get_all_current())

6. **PUT /pricing/crew-config** (Depends(require_admin)):
   - Accepts UpdateCrewConfigRequest (includes aircraft_type)
   - Calls crew_repo.create_version()
   - Returns new version

7. **POST /pricing/projects** (Depends(get_current_user)):
   - Creates project with auto-attached config version IDs.
   - Wiring for crew config FKs: Call `crew_repo.get_all_current()`, then filter the returned list to find the row where `aircraft_type == 'A320'` and use its `id` as `crew_config_a320_id`, and find the row where `aircraft_type == 'A321'` and use its `id` as `crew_config_a321_id`. Also call `pricing_repo.get_current()` and use its `id` as `config_version_id`.
   - Pass all three FK IDs to `project_repo.create_project()`.
   - Returns ProjectResponse

8. **GET /pricing/projects** (Depends(get_current_user)):
   - Lists user's projects

9. **GET /pricing/projects/{id}** (Depends(get_current_user)):
   - Returns project with its MSN inputs

10. **POST /pricing/projects/{id}/msn** (Depends(get_current_user)):
    - Adds MSN input to project

11. **PUT /pricing/projects/{id}/msn/{input_id}** (Depends(get_current_user)):
    - Updates MSN input

12. **DELETE /pricing/projects/{id}/msn/{input_id}** (Depends(get_current_user)):
    - Removes MSN input
